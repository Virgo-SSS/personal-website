Hey, I'm Virgo Stevanus

I'm a full-stack software engineer and I lean heavily toward the backend.

I've been building web applications and backend systems for over 4 years. Most of my work right now is in Node.js, Next.js, and Express, plus CMS integration using Directus. I can work across the full stack, but given the choice I'll always pick the backend side of a problem over the frontend. The APIs, the architecture, the parts users never see but everything depends on. I started out with PHP and Laravel for two years before moving into the JavaScript ecosystem. Not a bad starting point honestly, it forced me to think about structure early. I'm also a Bangkit Academy graduate, which is a Google-led program in Indonesia where I built my foundation in cloud computing and GCP. Right now I'm also doing real work with n8n and LLM APIs like OpenAI and Anthropic. I'm not going to oversell it, I'm still learning that space. But it's actual work I'm doing and a direction I want to keep growing in. Outside of work I've been going deep on Node.js internals and OS fundamentals. Mostly because I find it genuinely interesting to understand how things work, not just how to use them.
